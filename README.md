# react100-vstda

## Purpose of Program
Creates a very simple ToDo app with REACT.

- adds todo Item
- adds priority for todo Item
## Setup

### Clone
Clone respository
### Install
```
$ npm install
```
### Run
1. Open a browser at localhost: 3000

2. Build and start server
```
$ npm start
```
(npm run build and npm start combined in package.json with `npm start`.)
